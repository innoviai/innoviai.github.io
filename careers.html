Career

Full Stack Developer

Job Description:

-Working with the Design team to implement new features on both the front end and back end platform.
-Working with developers and project manager from UI/UX conceptualization to finished products and produce high quality interactive apps and mobile websites. The flow may involve a variety of processes from brainstorming ideas, playing around concepts, testing codes, inspiring designs; all freely, boldly and technically transformed into a meaningful user experience.
-Build new Front-end and Back-end UI templates for the web and mobile platform.
-Build highly scalable web applications, APIs and internal tools.
-Continuous integration/Continuous deployment (CI/CD pipelines) of Web application
-Design robust code structure to allow fast iteration of minimum viable products (MVP).
-(Optional) Improve frontend and backend performance, carry out A/B testing and help growing widely used products and services
-(Optional) Solution design and develop web application in Cloud Paas (AWS/Azure/Oracle)
-(Optional) Business Intelligence application integrations with vendors like PowerBI, Tableau, GoogleAnalytics to create dashboards and reports

Skills Required:

-Proficient with HTML5, CSS3, javascript, jQuery, CSS, Bootstrap.
-Hands-on experience in front-end framework & libraries (e.g., ReactJS/Angular/vue.js)
-Proficient with PHP/Java/Python/JSP/JSF/Node.js
-Hands-on experience in programming projects and knowledge of Git and Github/Bitbucket/Gitlab
-Familiarity with database technology such as MySQL or MongoDB and caching technology such as Redis/MemCache/Postgresql
-An advantage if familiar with Cloud Computing of Amazon AWS/Microsoft Azure/Google Cloud/GCP
-An advantage if familiar with Typescript, ECMAScript
-An advantage if familiar with MEAN/MERN Stack, Gridsome, Gatsby and Nuxt
-An advantage if familiar with graphql and Neo4j

Designer and Front-end Developer

Job Description:

-Work with designers to focus on new feature design and prototyping on retail and fashion industries.
-Build a beautiful, delightful experience for our customers.
-Collaborate with backend engineers, UX designers and ML Engineers.
-Build reusable code and libraries for future use
-Ensure the technical feasibility of UI/UX designs
-Optimize application for maximum speed and scalability
-Assure that all user input is validated before submitting to back-end
-Involve in discussion, planning and brainstorming for different digital products.
-Follow and improve our development process including stand-up meetings, repo and code review with github, issue tracking, etc.

Skills Required:

-Degree / Diploma holder of computer science, engineering, or related disciplines.
-Proficient experience of web markup, including HTML5, CSS3
-Proficient Experience in bootstraps or related web frameworks
-Good experience of client-side scripting and JavaScript frameworks, including VueJS or ReactJS is a Plus
-Basic knowledge of image authoring tools, to be able to crop, resize, or perform small adjustments on an image. Familiarity with tools such as as Gimp or Photoshop is a Plus.
-Proficient understanding of cross-browser compatibility issues and ways to work around them.
-Proficient understanding of code versioning tools, such as Git or GitHub is a Plus.
-Fast learner in a startup environment
-Good team player, self-organised, and willing to find new solution
